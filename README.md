This repository contains the frontend landing page prototype for Vitalynx, an AI-powered SaaS-based Hospital Management System created for the Masai Hackathon.

Project Overview
Vitalynx is designed to help hospitals digitally transform within 24 hours using AI-based assistance. Key features demonstrated via the landing page:
AI-powered assistance
Multi-hospital (multi-tenant) support
Role-based access approach
Simple and user-friendly UI

This repository contains the UI-only prototype.
